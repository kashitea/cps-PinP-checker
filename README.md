# PinP CPS Checker
**Picture-in-Picture** で使えるCPS測定器(クリック速度計測器)です。 

![スクリーンショット](./screenshots/screenshot1.png)

## 機能
- Picture-in-Picture機能を使ってゲーム中にもCPSが測定できます。
- 最大CPSの表示
